## [1.0.0] - 2026-04-10

### Added
- Initial release
- Security tab with present/missing indicators for 13 HTTP security headers
- Security score bar (percentage of required headers present)
- All Headers tab with search/filter
- Settings page with per-header checkboxes grouped by category
- Select All / Deselect All bulk actions in settings
- Copy to clipboard for any header value
- HTTP status code badge in URL bar
- Manifest V3 service worker architecture
- Dark theme UI
- Info-disclosure header warnings (Server, X-Powered-By)
